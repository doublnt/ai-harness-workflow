# Safety

## Repository writes

- Scan first.
- Draft before modifying existing prompt files.
- Ask before writing profile or enforcement files.
- Ask before installing local hooks.

## Secrets

Do not read `.env`, `.env.*`, credentials files, or secret directories.

## Domain examples

Examples are interview seeds only. They must not be applied as authoritative project rules.

## Enforcement

Hard enforcement is optional and must be explicit.
